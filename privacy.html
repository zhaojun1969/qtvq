隐私政策

更新日期：2026年6月4日 · 生效日期：2026年6月4日

「我心永恒 - Q问」（域名为 qtvq.cn，以下简称「本产品」或「我们」）由 我心永恒（北京）网络科技有限公司（以下简称「本公司」）运营。 我们深知个人信息对您的重要性，将按照《中华人民共和国个人信息保护法》《网络安全法》《数据安全法》等法律法规， 参照行业通用实践，保护您的个人信息与隐私安全。

一、适用范围

本政策适用于您通过网站 qtvq.cn、相关移动客户端、微信小程序、支付宝小程序、鸿蒙应用及我们不时提供的其他形态访问、使用本产品时的个人信息处理活动。若某端另有单独隐私说明，以该端说明为准；未约定部分仍适用本政策。

二、我们收集的信息及用途

1. 您主动提供的信息

  • 提问与对话内容：您在 Q 问中输入的文字、语音转写内容，用于生成 AI 答复与额度统计。
  • 会员与汇款信息：办理会员时您提交的汇款人姓名、汇款时间、金额、备注等，用于核对对公到账并开通服务。
  • 客服留言：通过 qtvq@qtvq.cn 或站内留言提交的文字、图片及可选邮箱，用于客服回复。

我们不要求您提供真实姓名、身份证号、手机号等敏感身份信息；请勿在提问或留言中主动填写上述信息及银行卡密码、验证码等。

2. 自动收集的信息

  • 设备标识（clientId）：由浏览器或客户端本地生成的匿名设备编号，用于 24 小时提问额度、会员状态同步，存储于本地与服务器 KV。
  • 本地存储数据：Q 币、智慧值、直线值、浏览记录、提问历史摘要等，用于产品功能与体验，主要保存在您设备本地（localStorage 等）。
  • 日志与网络信息：访问时间、请求路径、IP 地址（由接入层/CDN 记录），用于安全风控、故障排查与统计，通常保留合理期限。

3. 权限说明（移动端）

  • 麦克风（可选）：用于语音输入提问，仅在您主动点击语音按钮时使用。
  • 网络：用于访问 API 与加载内容。
  • 相册/相机（可选）：仅在上传客服截图时使用，由您主动选择。

三、Cookie 与同类技术

我们可能使用 Cookie、本地存储（LocalStorage）、SessionStorage 等同类技术，用于维持登录态（如有）、记住设备编号、保存偏好设置与额度缓存。您可通过浏览器设置清除本地数据；清除后部分功能（如免费额度计数）可能需重新计算。

四、第三方服务与委托处理

为实现 AI 问答、额度存储、备份与加速，我们可能接入以下第三方：

第三方用途可能处理的信息
Cloudflare（Workers / Pages / KV / AI）API 托管、AI 推理、额度存储提问内容、clientId、额度与会员状态
阿里云 OSS动态数据备份、国内读取加速脱敏后的 API 快照、额度备份（不含明文密码）
阿里云（服务器/Nginx）静态站点 qtvq.cn 托管访问日志

我们仅向第三方提供实现功能所必需的信息，并通过协议要求其依法保护个人信息。第三方处理规则亦受其自身隐私政策约束。

五、信息共享、转让与公开披露

我们不会出售您的个人信息。仅在以下情形可能共享或披露：

  • 取得您的单独同意；
  • 为履行法律法规义务、司法或行政机关依法要求；
  • 为保护您、我们或公众的人身、财产安全所必需；
  • 公司合并、分立、收购时，依法告知并继续受本政策约束。

六、存储地点与期限

  • 主要业务数据存储于 Cloudflare 及/或中华人民共和国境内服务器与对象存储。
  • 提问与额度数据：在实现目的所需期限内保存；客服留言备份约 90 天；日志按合规要求留存。
  • 超出期限后，我们将删除或匿名化处理，法律法规另有规定的除外。

七、信息安全

我们采取 HTTPS 传输、访问控制、密钥分级管理、备份与审计等措施保护数据安全。尽管已尽合理努力，互联网环境无法保证绝对安全；请您妥善保管设备与账号，勿泄露汇款核实密钥等工作人员凭证。

八、您的权利

根据适用法律，您享有以下权利（行使方式见下文「联系我们」）：

  • 查阅、复制：了解我们处理的与您相关的个人信息；
  • 更正、补充：发现信息有误时可申请更正；
  • 删除：在法定情形下请求删除；您也可自行清除浏览器本地数据；
  • 撤回同意：撤回语音、相册等权限授权（在系统设置或浏览器中操作）；
  • 注销：客户端上线后将提供账号注销入口；目前 Web 端可联系我们协助删除服务端 clientId 关联数据；
  • 投诉举报：向网信、公安等主管部门投诉。

我们将在15 个工作日内响应您的合理请求；需核实身份后处理，法律法规另有规定的除外。

九、未成年人保护

本产品主要面向成年人。若您为未满 14 周岁 的未成年人，请在监护人指导下阅读本政策并使用服务。若您为未满 18 周岁 的未成年人，请在监护人同意下使用付费功能。我们不会主动面向儿童收集个人信息；若发现未经同意收集，将尽快删除。

十、政策更新

我们可能适时修订本政策。重大变更将通过网站公告、弹窗等显著方式提示。若您继续使用服务，即表示接受更新后的政策；若不同意，请停止使用并联系我们。
